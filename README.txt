MemeStar by Shaham Tamir

MemeStar is a browser-based meme generator that lets users create,
customize, and share memes effortlessly. With an intuitive interface,
you can select from a range of meme templates or upload your own images,
add and style text, and export your creation for sharing.

Features:
- Easy meme creation and customization.
- Responsive canvas using the HTML5 Canvas API.
- Direct sharing to social platforms.
- Local storage for saving drafts.

Technologies: HTML5, CSS3, JavaScript, Canvas API.
